# Simple Parser In C++
## Supported operators (sorted in descending order of calculation priority)
<p>
()            | change priority <br>
!, -, +       | unary<br>
^             | exponentiation<br>
*, /, %       | multiply, division, remainder<br>
+, -          | addition, subtracting<br>
\>, >=, <, <= | logical expressions<br>
==, !=        | equal, unequal<br>
&&            | and<br>
||            | or<br>
=, +=, -=, %= | assignment<br>
\*=, /=, ^=   | ^^<br>
