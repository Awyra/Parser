# Simple Parser In C++
## Supported operators (sorted in descending order of calculation priority)
|Operator|Short Description|
|:-----|:-----|
|() | change priority|
|!, -, + | unary|
|^ | exponentiation|
|*, /, % | multiply, division, remainder|
|+, - | addition, subtracting|
|\>, >=, <, <= | logical expressions|
|==, != | equal, unequal|
|&& | and|
|&#124;&#124; | or|
|=, +=, -=, %=, \*=, /=, ^= | assignment|
